# laravel-angular

## Laravel 5 Rest API

### Installation

* **Install PHP, MySQL, Composer**
* **Install Dependencies**
  `cd server`
  `composer install`
* **Database Migration**
  `php artisan migrate`
* **Laravel Key Gernation**
  `php artisan key:generate`
* **Serve App**
  `php artisan serve`
  
## Angular 5 Client App

### Installation

* **Install Node, Npm**
* **Install Dependencies**
  `cd client`
  `npm install`
* **Serve App**
  `npm start`
